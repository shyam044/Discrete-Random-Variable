Discrete Random Variable

Definition

A random variable is called discrete if its range is either finite or countably infinite.

Probability Mass Function (PMF)

Σ P(x) = 1    and    0 ≤ P(x) ≤ 1

Mean or Expectation

E(X) = Σ x × P(x)

Variance

Var(X) = E(X2) - [E(X)]2

Standard Deviation

SD(X) = √Var(X)

Uniform Discrete Random Variable

P(X = x) = 1/n

E(X) = (a + b) / 2

Var(X) = (n2 - 1) / 12

Tips:
  • Use PMF to define probabilities of each possible value of a discrete variable.
  • Expectation gives the average value in the long run.
  • Standard deviation and variance measure the spread of values.
